# Mi Primera Página Web Personal

## Descripción

Página web personal desarrollada como proyecto integrador para la materia de Programación de Sistemas Web en la Universidad Ecotec.

## Autor

Diego Benavides Luna - Ingeniería en Sistemas

## Secciones

- **Inicio**: Encabezado con nombre y menú de navegación
- **Sobre mí**: Información personal y profesional
- **Intereses**: Lista de hobbies y gustos
- **Formación**: Tecnologías actuales y metas de aprendizaje
- **Contacto**: Formulario de contacto

## Tecnologías utilizadas

- HTML5
- CSS3
